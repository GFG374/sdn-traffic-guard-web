# RAG知识库使用指南

## 问题描述
您已经成功上传了文件到系统，但在询问AI分析文档内容时，AI仍然提示您上传文件。这是因为AI系统需要明确的指令来访问知识库中的文档。

## 解决方案

### 方法1：智能RAG自动触发（推荐）
上传文件后，系统会自动使用RAG知识库，无需特殊关键词：

1. **直接提问即可**
   ```
   请分析我上传的网络安全文档
   ```

2. **自然语言提问**
   ```
   网络监控的最佳实践有哪些？
   如何根据文档优化安全策略？
   ```

3. **可选的显式触发方式**
   - 仍支持 `/rag` 前缀或"使用知识库"关键词
   - 适用于需要明确指定知识库查询的场景

### 方法2：文件上传时添加到知识库
在上传文件时，系统会询问"是否将文件添加到知识库用于RAG查询？"，选择"是"即可确保文件可用于后续分析。

### 方法3：使用界面功能
1. 点击上传按钮上传文档
2. 在输入框中使用上述任一触发方式提问
3. AI将基于知识库中的文档内容回答

## 示例对话

**错误用法：**
```
请分析我上传的文档
```
（系统会调用普通AI，无法访问知识库）

**正确用法：**
```
/rag 请分析我上传的网络配置文档并总结最佳实践
```
（系统会调用RAG API，访问知识库中的文档）

## 验证方法

您可以测试以下查询来确认RAG功能正常工作：

1. `/rag 知识库中有哪些文档？`
2. `使用知识库，网络监控的关键指标有哪些？`
3. `根据文档内容，如何设置网络安全策略？`

如果RAG功能正常工作，AI的回复会包含"参考文档"部分，列出使用的具体文档名称。
